📘 高一会计重点整理（考试快速复习）

本页面用于考试前快速回顾所有核心概念。

🟦 第一章：会计基础

  • 会计 = 记录 + 分类 + 总结 + 报告
  • 目的：提供信息辅助决策
  • 使用者：内部（老板）+ 外部（投资者/银行）
  • 概念：
    • 经济实体假设
    • 持续经营
    • 会计期间
    • 历史成本
    • 配比原则

🟦 第二章：资产、负债与权益

  • 资产：公司拥有的资源
  • 负债：公司欠别人的钱
  • 权益：资产 - 负债
  • 会计等式：资产 = 负债 + 权益

🟦 第三章：借贷记账法

  • 每一笔交易：借 = 贷
  • 借方：
    • 资产增加
    • 费用增加
  • 贷方：
    • 负债增加
    • 权益增加
    • 收入增加
  • 分录格式：
    借：xxx
    贷：xxx

🟦 第四章：分类账

  • 分类账 = 每个账户的记录
  • 从日记账过账（Posting）到分类账
  • T账户：
    • 左 = 借
    • 右 = 贷
  • 用途：汇总账户余额

🧠 高频考试重点

  • 会计等式（一定要会）
  • 借贷规则（最常考）
  • 分录（journal entry）
  • 资产/负债/权益分类
  • 过账（posting）
  • T账户结构

⚡ 快速记忆口诀

  • 资产费用 → 借增加
  • 负债权益收入 → 贷增加
  • 借 = 贷（永远平衡）
  • 资产 = 负债 + 权益

🎯 考试技巧

  • 先判断交易影响哪个账户
  • 确认是增加还是减少
  • 再决定借或贷
  • 最后检查借贷是否平衡
