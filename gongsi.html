📘 会计公式整理（考试重点）

本页面整理所有高一会计必须掌握的重要公式。

🟦 1. 基本会计等式

资产 = 负债 + 权益
  • 最核心公式
  • 所有会计记录都围绕此等式

🟦 2. 权益计算公式

权益 = 资产 - 负债
  • 用于计算所有者权益
  • 常见考试计算题

🟦 3. 利润计算公式

利润 = 收入 - 费用
  • 收入 > 费用 → 盈利
  • 收入 < 费用 → 亏损

🟦 4. 资本变动（扩展）

期末权益 = 期初权益 + 利润 + 额外投资 - 提款
  • 适用于更进阶题目

🟦 5. 折旧公式（基础）

年折旧 = (成本 - 残值) ÷ 使用年限
  • 用于固定资产
  • 常见计算题

🟦 6. 现金余额计算

期末现金 = 期初现金 + 收入 - 支出
  • 用于现金流概念

🧠 总结（考试重点）

  • 资产 = 负债 + 权益（最重要）
  • 权益 = 资产 - 负债
  • 利润 = 收入 - 费用
  • 折旧 = (成本 - 残值) ÷ 年限
考试技巧：
看到题目先判断是“资产、利润还是权益”再选公式。
